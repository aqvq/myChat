## Python多继承中，使用super调用父类带参数初始化函数的问题

一种较为简便的解决办法
```python
class A(object):
    def __init__(self, *args, **kwargs):
        pass

class B(A):
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
    
class C(A):
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self._name = args[0]
        print(self._name)
    
class D(B, C):
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)

d = D("Jiaqi")

```
